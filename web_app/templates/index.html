Теория метода

Метод Монте-Карло для вычисления π основан на вероятностном подходе:

  1. Рассматриваем квадрат со стороной 2 и вписанный в него круг радиусом 1
  2. Отношение площадей: Sкруга/Sквадрата = π/4
  3. Генерируем случайные точки в квадрате
  4. Отношение точек в круге к общему числу точек ≈ π/4
  5. π ≈ 4 × (точки в круге) / (все точки)

π ≈ 4 × 0 / 0 = 0
